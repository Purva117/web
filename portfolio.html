Global News Navigator

Global News Navigator provides a dynamic, interactive summary of global news from the past 24 hours. Utilizing a sophisticated language model, the platform aggregates news headlines and categorizes them by geographic location. Each country's news is summarized, highlighting key updates and overall sentiment. Users can explore this information through an engaging 3D globe interface, allowing them to visually interact with the latest global developments and understand the prevailing sentiment in each region. Stay informed and connected with the world like never before with Global News Navigator.

SmartStock Predictor

SmartStock Predictor leverages historical stock price data since 2000 and employs LSTM models to forecast stock prices for the upcoming month. It provides users with buy/sell recommendations and displays both actual and predicted prices on an interactive chart, along with performance statistics.

Marvel Character Network

Marvel Character Network is an interactive visualization of MCU characters, showcasing their relationships and characteristics. Created using D3.js, this project allows users to explore the Marvel Universe through dynamic nodes representing each character. View detailed information, including network centrality stats and movie appearances. Dive deep into the Marvel Universe and discover intricate character connections with Marvel Character Network.

GeoViz Landscape Narrator

GeoViz Landscape Narrator provides an attractive and detailed map of any location within a 15 km radius, highlighting building structures, green areas, and water bodies. Using Python libraries OSMnx and NetworkX, this tool can extract and visualize road networks, selective green fields, and water bodies based on provided coordinates. Users can easily change the location and radius in the Main.ipynb file, with Dresden, DE as the default example. Explore and visualize urban landscapes with GeoViz Landscape Narrator.
